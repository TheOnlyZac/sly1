◆ SetSensorSensors()

void SetSensorSensors ( SENSOR *  psensor,
SENSORS  sensors 
)

Sets the sensor's state and handles triggering the sensor's alarm.

Parameters
psensorPointer to the sensor.
sensorsNew sensor state.

If the sensor is transitioning from SENSORS_SenseEnabled to SENSORS_SenseTriggered, it will trigger the associated alarm if one is set. If the sensor's state changes, it will also notify any listeners of the change via an LO splice event.
