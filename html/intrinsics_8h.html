Read MIPS CP0 Count register (register $9).

The Count register increments automatically every CPU cycle and is used for high-resolution timing. On PS2 EE, it runs at 294.912 MHz. https://psi-rockin.github.io/ps2tek/#eecop0timer

Returns
32-bit Count value as unsigned long

Hardware square root using MIPS FPU sqrt.s instruction.

Inline assembly to invoke the standard MIPS II FPU sqrt.s instruction. Available on R5900 (PS2 EE) and most MIPS processors with an FPU.
