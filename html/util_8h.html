Evaluates a quadratic curve at parameter x.

Computes the quadratic equation: g0 + x*g1 + x²*g2 Uses Horner's method for efficient evaluation: g0 + x*(g1 + x*g2) This reduces multiplications from 3 to 2 compared to the expanded form.

Parameters

xParameter value
Returns
Evaluated curve value at x
